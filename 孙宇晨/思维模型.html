═ 先 看 一 下 你 自 己 ═

你以为孙宇晨在玩你看不懂的游戏？
不，你早就在玩底层一样的游戏。

下面 3 件事，你大概率做过其中 1-2 件。
▼ 你做过 换工作前，先把新 offer 谈到手 / 拿到 verbal，
再开口辞职。
→
▲ 孙宇晨在做 同时跑两条赛道（A 给现金流，B 攒真正的赌注），
等 B 起来，把 A 的资源一次性导进 B。
▼ 你做过 副业用马甲号 / 用一个有限责任公司收款，
不挂自己实名头像。
→
▲ 孙宇晨在做 把"个人"和"公司主体"物理切割。
出事让公司扛，个人不被连坐。
▼ 你做过 抢早鸟价 / 抢茅台 / 抢演唱会票 / 抢小米 SU7 首批，
朋友圈晒一下。
→
▲ 孙宇晨在做 花 $457 万抢一顿和巴菲特的午餐。
同样是抢稀缺事件，他抢的能上头版。
你看，你早就懂这套底层语法了。
区别只是—— 孙宇晨把这 3 件做到了 100 倍尺度，
再用另外 3 件你学不来的事把它放大。
✅ 这 3 个你能学 M1 / M4 / M6 · 普通人能等比例缩小到自己尺度
M01 · 能 学
注意力套利
—— 你早就懂"抢稀缺"，他只是抢的尺度更大

🪞你已经在用的版本

你抢早鸟价、抢茅台、抢演唱会票、抢小米 SU7 首批用户身份—— 你不只是为了那个东西，你也是为了"被人看到你抢到了"。 发朋友圈、晒社群、写小红书，都是顺手的。

这就是注意力套利的微缩版：用一笔钱（或一段时间）同时换两件东西——商品本身 + 你被记住。 多数人不会算第二件东西的价值，但你下意识在用。

🚀孙宇晨怎么用到 100 倍

他不是不算第二件东西的价值，是把第二件东西当主菜—— 商品本身只是入场券。

2019.06 $457 万拍下巴菲特慈善午餐，破当时 19 年最高价。换来全球财经媒体头版数月，单位曝光成本远低于投广告。
2024.11 $624 万买下"用胶带粘在墙上的香蕉"艺术品，当众吃掉。这事估价只值 100-150 万，剩下的钱就是注意力定价。
2025.05 花约 $1900 万买够 TRUMP 币，坐稳特朗普私人晚宴第一名席位，拿到限量陀飞轮腕表合影。

✅你怎么用

  1. 今天 列出你打算未来 3 个月做的 3-5 件事。在每件事旁边写一句：「这件事会让谁专门搜我的名字？」 答不上来的，就是只能换钱、换不了注意力的事。
  2. 这周 挑一笔你本来就要花的"获客预算"（投流 / 信息流 / 朋友圈广告 / 拓展礼）—— 把它从"投放"改成"做事件"。同样的钱，做一件别人会主动谈论的事。
  3. 这个月 做一次"反共识下注"—— 在你的圈子里，公开做一件 80% 的人觉得不值 / 不会做的事（金额可以小，但姿势要鲜明）。这件事的回报不是事本身，是它带来的搜索量和讨论量。
  4. 每季度 复盘一次：过去 90 天，"我做的事 vs 我说的话"，哪个带来的关注更多？多数人会发现是前者。如果是后者，你还在做"内容创作者"，没在做"事件创作者"—— 这两件事的天花板不一样。

⚠️你和他不一样的地方

他能这么搞 18 年不死，是因为他有"道德库存"——每用一次就消耗一点，但他底子厚。 普通人没那么厚的库存，三五次就会被打回原形（"哦原来他每次都这样"）。 所以你要的不是"每件事都搞个大新闻"，是每年精选 1-2 件做大—— 频率低、声势集中。

M04 · 能 学
双轨并行 + 一次性切换
—— 你早就懂"先把 offer 拿到手再辞职"，他只是把这套放到了 5 年尺度

🪞你已经在用的版本

换工作前你不会先裸辞——你会同时跑两条线：现在的工作给现金流，新机会偷偷面试。 等新 offer 到手、待遇确认、入职日定下来，你才开口辞职。

这就是双轨并行 + 一次性切换。大多数人本能就会做这个—— 因为切错了代价太大。 但很多人会停在 "短期换工作" 这个尺度，没把它放大到 1-3 年的事业切换尺度。

🚀孙宇晨怎么用到 100 倍

他不是换工作—— 是同时跑两个表面无关的赛道，让 A 给 B 做"资源储备池"， 等 B 起来，一次性把 A 的全部资源导进 B，然后弃掉 A。

2014-2017 表面赛道：陪我 APP（社交产品，跑现金流和 6000 万 A 轮融资 + 福布斯 / 湖畔大学光环）；地下赛道：默默搭建波场 TRON（一个加密货币项目）。
2017.08 波场做 ICO（加密币首次发行，类似公司上市但用的是币）时，分配方案里把 10% 直接给"陪我（北京）科技"—— 把 A 赛道的用户和现金流，一次性导进 B 赛道。
2018-2019 波场起飞、单次套现 $3 亿之后，陪我 APP 注销。表面赛道弃用，地下赛道独立运行。

✅你怎么用

  1. 今天 列出你正在做的 1-2 件主事（你的本职 / 主要收入来源）。 这是你的"主轨"—— 它要继续跑，不是用来弃掉的。
  2. 这周 在主轨之外，开一条"看似无关、但能切换过去"的副轨。 关键词是"看似无关"—— 如果副轨和主轨完全同构，它救不了你；如果完全无关，又导不过去。 理想是「同一批人对它感兴趣，但形式不同」（比如：本职是写代码 → 副轨是教别人写代码 / 做 AI 工具 / 写技术博客）。
  3. 这个月 给副轨设最低投入—— 比如「每周 6 小时 + 每月 ¥500 试错预算」。 少于这个就停了，多于这个会侵蚀主轨。这个数字要写下来贴墙上，不是放在心里的。
  4. 12 个月 给副轨设"切换信号"—— 客观、可测的那种。比如：「副业 MRR 连续 3 个月 ≥ 主业月薪 60%」 或 「订阅数过 5000 + 转化率 ≥ 2%」。 信号不到，绝对不切。多数人切失败不是没本事，是切早了。
  5. 12 个月 也给副轨设"退出信号"—— 比如「投入满 18 个月，订阅数仍 < 500 / MRR 持续 ¥0」。 双轨不是"骑墙一辈子"，是主动设计的过渡结构。退出信号触发就果断停—— 砍掉副轨比砍主轨容易得多。

⚠️你和他不一样的地方

孙宇晨能"一次性切换"是因为主轨现金流足够厚—— 陪我有 6000 万 A 轮和持续打赏收入。 普通人切换前必须先攒够 6-12 个月生活费作缓冲，否则切换信号触发那一刻你也不敢动手。 现金流不够 ≠ 不切，而是把"切换信号"门槛设高一点，多跑几个月再说。

M06 · 能 学
主体切割 / 风险绝缘
—— 你早就懂"副业用马甲号"，他只是把这套放到了法律和资金维度

🪞你已经在用的版本

副业你不会用实名头像，会另开一个号。 在闲鱼出二手你也不挂工作邮箱。 贷款买房之前你会查信用记录、避开过度负债。

这些都是同一件事——把"做这件事的我"和"作为整体的我"切开， 让任何一头出问题不会连坐到另一头。这条对个人开发者 / 副业玩家反而最实用， 因为你接触外部用户和钱越多，被连坐的风险越大。

🚀孙宇晨怎么用到 100 倍

他把这套放到法律主体维度—— 公司、基金会、外国实体、辞 CEO，全都是同一个动作： 让法律责任和资金责任找不到唯一收件人。

2017.07 波场 ICO 前 1 个月，先卸任陪我 APP CEO。把"波场创始人"和"陪我 CEO"两个身份提前切开。
2019.07 陪我 APP 被新华社点名"涉黄音视频"—— 公司决议直接解散，业务消失，个人无追溯。
2026.03 SEC（美国证监会）案最终和解：Rainberry 公司付 $1000 万罚款，孙宇晨本人零处罚且不予重提。这就是主体切割打到极致的样子。

✅你怎么用

  1. 今天 盘一遍你目前有外部用户付费的副业 / 接私活 / 卖课卖咨询—— 收款是不是用的个人账户、个人微信、个人 PayPal？ 有一个就有一个主体切割漏洞。
  2. 这周 上工商网站查一下：在你所在地，注册一家小微企业 / 个体工商户 / 一人有限公司的真实成本是多少。 答案大概率比你想的便宜（中国大陆 ¥0-2000；香港 / 新加坡 / 美国 LLC 几千到一万）。
  3. 这个月 凡是有外部用户付费的副业，全部换到那个法律主体下走账。 收款、合同、发票、退款、纠纷—— 全部走主体，不再走个人。 这是孙宇晨用了 18 年最稳定的一条防御。
  4. 这个季度 做一份"个人 vs 业务"清单：哪些资产 / 账号 / 邮箱 / 手机号是「个人不能丢的」，哪些是「业务出事可以一起埋掉的」。 清单写下来，定期检查这两堆有没有混。混了就拆。
  5. 长期 副业做大到一定规模（年收入过 ¥50 万 / 涉及多个客户合同），找一次1-2 小时的律师 / 会计咨询。 几千块换一次专业意见，比出事后赔几十万划算 100 倍。

⚠️你和他不一样的地方

主体切割只在"法律 / 资金"维度有效，在"声誉"维度无效。 SEC 案最后确实公司付罚款、孙本人零处罚，但外界对他的"个人即责任主体"判断没消失。 普通人也一样—— 注册公司能挡住法律连坐，挡不住圈子里的口碑传播。 所以这条不是"出事就改名"的免死金牌，是"提前隔离财务和法律风险"的防御工事。

🚫 这 3 个别学 M2 / M3 / M5 · 普通人模仿 = 大概率违法 / 信用清零 / 被秒杀
M02 · 不 建 议
监管 / 法律辖区套利

他做的事是：同时持有多个国家的身份/居留权（圣基茨、马耳他、格林纳达大使）， 在一个辖区监管收紧时立刻切换到另一个辖区放大业务。 这需要千万美元起的资本量级 + 对各国监管时差的实时把控 + 律师团队 + 政治人脉。 普通人尝试 = 大概率被中介骗投资移民、或在不熟的辖区违法。

🟡 普通人版的最弱替代 想类似的事，最多做"区域套利"—— 找新一线城市的落户补贴、 跨境电商的窗口期、新平台的早期创作者红利、海外远程工作的汇率差。 绝对不要碰跨国监管套利、不要为"避税"去注册不熟悉的离岸公司。
M03 · 不 建 议
机构背书寄生

他做的事是：拿一个短期、低门槛的与高声誉机构的关联（实习、约稿一篇、校级课题挂名）， 在简历 / 名片 / 媒体稿里反复用 5-10 年。 这套打法在 2010 年代有效，但AI 时代普通人 1 分钟就扒得动你所有履历—— LinkedIn 标题虚标、自封的 title、没坐实的"客座教授"，识破速度是 10 年前的 100 倍。 有效期已从 10 年压缩到 6 个月。

🟡 普通人版的最弱替代 每年争取 1 次"高声誉平台的可证明关联"—— 一次大会发言、一次有名媒体投稿、 一次开源项目的 contributor、一次 YC / 字节 / OpenAI 校友活动出席。 关键是「可证明」—— 别人随手一搜能找到证据。 虚标的成本在 AI 时代远高于诚实的成本。
M05 · 不 建 议
否认范式 + 切换战场

他做的事是：被指控时不修复、不道歉、直接否认，然后迁移到一个不在乎这个错的新圈子。 他能切是因为他每个圈子都有备份赛道—— 写作圈出局可以跳加密圈，加密圈出问题可以跳政治圈。 普通人没有"下一个战场"可切—— 你的同事、客户、朋友都在同一个微信生态里， 在小圈子里硬否认 = 信用清零，再无翻身。

🟡 普通人版的最弱替代 出事时不要默认"必须道歉到底"，先冷静评估"修复 vs 切换"的成本。 如果是小事故，诚实坦白 + 修复永远是首选（成本低、还能赢回信任）； 如果是关系彻底破裂，体面退出 + 时间换空间，比硬扛 / 硬否认强 100 倍。 核心是：永远不要把"否认"当默认动作。
═ 一 句 话 总 结 ═
孙宇晨的真实公式不是"包装 + 营销 + 投机"，
是 「注意力放大 + 多注下注 + 主体绝缘」。
前者是外界对他的标签，后者才是从 14 个决策节点里反复跑出来的动作结构。
后两件，才是普通人能学的部分。
