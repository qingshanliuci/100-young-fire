# 100 个财富自由的年轻人

> 在 AI 时代，路径不可复制，但思维模型可迁移。

## 这是什么

100 个真实案例的深度档案库。不写"成功学故事"，只做两件事：

1. **事实层**：用 7 个维度的公开材料交叉验证，建立经得起推敲的人物时间线。
2. **抽象层**：从事实里蒸馏出可迁移的"思维模型"——他们怎么做决策、相信什么、为什么从那里开始。

每个人物都有：
- 一份**人生路线图**（事实层 HTML，扫一眼能看完）
- 一份**思维模型**（抽象层 HTML，蒸馏出 OS 三层架构）
- 七份**研究材料**（带来源 URL 和可信度标签的原始档案）

## 为什么要做这个

2014 年加入字节、2013 年买入 BTC 这样的路径都是一次性的。你没法复制——时机已经关闭。

但他们 12 年里反复使用的**那套思考方式**，可以照搬到任何时代。AI 时代尤其如此：路径在以更快的速度变化，但"以终为始"、"够代替最大化"、"环境 > 努力"这样的元规则不会过期。

100 个案例的目的，是让你看完之后，从第 N 个人身上抽出一条原则、第 M 个人身上抽出另一条，组合成你自己的 OS。

## 项目方法论

每个档案的制作流程都遵循同一份 SOP：[档案制作流程.md](档案制作流程.md)

核心原则三句话：

1. **三段式不能跳也不能混**：原料采集 → 交叉验证 → 事实层 HTML → 抽象层 HTML
2. **抽象层必须用反常识对比验证**：每条原则能写出"默认做法 vs 本人做法"两栏 = 真洞察；写不出 = 金句，删掉
3. **必须保留 caveat**：明确区分"本人明说" vs "从行为归纳"，不冒充本人认证
