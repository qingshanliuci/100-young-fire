# 05 关键决策链

## 采集摘要

郭宇（1991年生于江西）从行政管理专业自学编程转行，经支付宝实习、创业公司被收购进入字节跳动（2014-2020），28岁凭期权+投资实现财务自由后宣布退休。此后移居日本经营温泉旅行品牌，2022年进入Web3创业间隔年，2024年搬至硅谷转型风险投资人。其决策链的核心逻辑是"关注自己、做想做的事"，每次转折都是主动选择而非被动漂流。

## 核心发现

### 决策 1: 行政管理专业转行自学编程

- **时间**: 2008年（高考后）
- **背景**: 郭宇从江西矿山小城考入广州暨南大学，专业为政治与行政管理。家庭经济压力大——父亲患心血管疾病卧床，他需要找到能养活自己的出路。
- **动机**: "学编程是因为当时没有其他路可以走，因为我读的专业以后没有办法养活自己，所以必须得找到一条路。"此外，高考后他想为高中读书社建一个网上论坛，小姨和舅舅买了一台acer笔记本，性能不适合玩游戏却适合写代码，由此入门。一位在硅谷工作的阿姨给他指了"写代码"这条路。
- **过程**: 2008年纸质教材极少，没有网课和培训班，完全靠网上搜索自学、试错纠正。大学期间不怎么上课，在暨南大学勤工俭学招聘板上接外包项目赚钱，逐步积累实战经验。
- **结果**: 大二投递腾讯实习未进终面，大三写邮件给支付宝获得实习机会。编程能力在大学期间快速成长，为后续职业发展奠定基础。
- **本人反思**: "为了抵抗生命的重力，为了支撑自己未来的人生，我无路可走。""如果能回到十几年前，我想当作家。"他承认当时放弃了写作等兴趣来迎合生存需求，如今"又回到那个点"寻求补偿。
- **来源**: 36氪专访《"我不是富二代"：一位28岁退休程序员的自述》；21财经专访
- **可信度**: 一手（本人亲述）

### 决策 2: 加入支付宝实习并留任

- **时间**: 2010年大三（实习），2011年毕业后正式入职
- **背景**: 非计算机专业出身，仅凭自学编程能力求职。当时的互联网行业对非科班人才相对包容。
- **动机**: 需要一份能养活自己的工作，同时对技术有热情。"招他进来的那个人也不是计算机专业，是学英语的。那个时代非科班出身、但技术厉害的人非常多。"
- **过程**: 给支付宝写了一封实习求职信，对方打电话问了几句就让过去实习了。实习时拿到offer，毕业后留任。在支付宝实习期间月薪1500元。
- **结果**: 2011-2013年在支付宝任前端开发工程师，接触到优秀技术团队，"每个人都朝气蓬勃，认真钻研技术"。同时在2013年4月首次接触到比特币（通过Hacker News上比特币突破100美元的讨论），并在支付宝内部技术分享会上讨论。
- **本人反思**: 认为这是"幸运垂青"的一刻，支付宝"招人不论背景，只要能用就行"。
- **来源**: 36氪专访；罗磊博客
- **可信度**: 一手（本人亲述）+ 好友佐证

### 决策 3: 离开支付宝，北京创业

- **时间**: 2013-2014年
- **背景**: 在支付宝工作两年后，选择离开稳定工作去北京创业。先加入糗事百科，后参与"多说"评论系统开发。
- **动机**: 追求更大的技术挑战和创业理想。当时正值中国互联网创业热潮。
- **过程**: 从杭州到北京，创业公司一共三个人，在中关村小区租了两室一厅。期间"睡过半年的办公室地铺"。同时在GitHub上大量贡献开源项目，2013-2014年间GitHub活跃度位列中国区TOP3-5。
- **结果**: 创业公司（多说/图虫）最终被字节跳动收购，由此"稀里糊涂"进入字节跳动。
- **本人反思**: 这段经历虽然辛苦，但为加入字节跳动创造了机缘。他在这个阶段积累了大量开源声誉，2014年全球软件开发大会曾邀请他分享前端开发经验。
- **来源**: 36氪专访；罗磊博客《如何看待年仅28岁的郭宇宣布从字节跳动退休？》
- **可信度**: 一手（本人亲述）+ 好友佐证

### 决策 4: 加入字节跳动（被收购方式）

- **时间**: 2014年
- **背景**: 创业公司被字节跳动收购，当时字节跳动估值约5亿美元，员工约300多人，研发工程师不到100人。
- **动机**: 并非主动选择，而是被收购后顺势加入。"加入公司时是300多人，也不算初创员工。"
- **过程**: 2014年夏天搬入知春路的盈都大厦。作为早期员工获得了较为慷慨的期权分配——当时字节现金流紧张，给员工期权相对大方。
- **结果**: 2014-2020年在字节跳动工作6年，晋升至"资深技术专家"（高级首席工程师）。参与了小程序平台运行技术、"新海诚滤镜"（火到越南韩国日本）、时光相册、百万英雄等项目。工作模式为996甚至更高强度，"经常凌晨三四点才离开公司""每两周才过一个周末"，导致长白头发。期间字节跳动估值从5亿美元涨到约1000亿美元（约200倍）。
- **本人反思**: "期权确实是我原始积累的重要组成部分。"他深知自己无法胜任管理岗位，选择走技术路线。对字节文化中的"Always Day 1"理念表示认同。
- **来源**: 36氪专访；China Daily英文采访；搜狐新闻
- **可信度**: 一手（本人亲述）

### 决策 5: 投资加密货币（早期比特币购买）

- **时间**: 2013年4月首次接触，持续至今
- **背景**: 在支付宝工作期间，被Hacker News上比特币突破100美元的热议所吸引。
- **动机**: 技术好奇心驱动，作为程序员对去中心化技术有天然兴趣。
- **过程**: 2013年购买比特币，成为最早期使用者之一。之后长期持有，从未清仓。2017年ICO热潮时首次尝试编写智能合约。
- **结果**: 早期比特币持有带来了巨大的财富增值（2013年100美元到后来数万美元），成为其财务自由的重要组成部分之一。此外还定投互联网公司股票（腾讯等），2019年套现。
- **本人反思**: 2025年他认为"比特币的故事到此已经进入了终章，四年的周期已经结束"，"比特币已经失去了它设计初衷的意义：帮助每个人抵抗无限制膨胀的债务"。但他仍保留早期加密资产，未变现。
- **来源**: PANews专访（2025年10月）；Twitter/X @turingou；Mirror.xyz
- **可信度**: 一手（本人亲述）

### 决策 6: 28岁宣布"退休"离开字节跳动

- **时间**: 2020年2月（微博发布离职信）
- **背景**: 在字节工作6年，已晋升至资深技术专家。日本房产的租金收入+其他被动收入已与工资打平。工作已步入正轨，形成交接契机。
- **动机**: 多重因素叠加：(1) 被动收入已等于工资；(2) 自认"贡献了自己能贡献的价值，完成了想做的事情"；(3) 无法也不想走管理路线；(4) 有更想做的事——温泉旅馆、写作、旅行等人生清单。原文："我选择在28岁的末尾退休，拥抱山间清泉与峡谷的风，去感受春秋冬夏。"
- **过程**: 提交辞职信后在微博发布退休宣言。4个月后被人翻出并爆红网络，登上热搜。
- **结果**: 引发全网关于FIRE（财务独立提前退休）的讨论。1600人加他好友问"挣了多少钱"。退休后迁居日本。
- **本人反思**: "所谓的28岁退休，于我而言，只是从互联网行业退休。""以往我觉得财务自由就是绝对的自由，其实是一个错误的判断。"强调："自由是个人的私事。钱多钱少，开心到老，想清楚什么让自己得偿所愿，才是最应该做的事儿。"
- **来源**: 36氪专访；21财经专访；PANews专访（2025）
- **可信度**: 一手（本人亲述）

### 决策 7: 移居日本 + 经营温泉旅行品牌

- **时间**: 2019年开始布局（在职期间），2020年退休后正式迁居
- **背景**: 在字节工作期间，实行双周末制，每个双周末飞去日本旅游、投资房产。退休前已去过日本100多次，走访530+温泉旅馆。自学日语考过N1。
- **动机**: "如果一个人能承受在北京购入一处房产，那么就没有他不可承受的旅行开支。"在温泉旅行中结识了通过日本房地产获得稳定被动收入的朋友，受到启发。想做"一个顶级的温泉旅馆"并融入中国特色。
- **过程**: 在东京购置多处房产（包括两套市中心物业、数栋三层楼建筑），年化收益7%-10%。2019年成立"株式会社山月夜"旅行社。退休后全职经营，为中国游客提供温泉旅行服务，同时为上海杂志撰写温泉旅行文章。
- **结果**: 房产租金收入达到每月数万元（等同于字节工资水平）。疫情对旅行社业务造成冲击，但房产投资收益稳定。退休后两个月购入宾利欧陆GT。
- **本人反思**: "旅行像洗澡一样可以激发灵感，它最大的意义是让我自省。"在日本享受"隐形人"的状态。
- **来源**: China Daily英文报道；YouTube《我住在这里的理由》第367期；Apple Podcasts《一起散步去》Vol.7
- **可信度**: 一手（本人亲述）+ 媒体实地采访

### 决策 8: 进入Web3创业间隔年

- **时间**: 2021年11月开始研究，2022年1月正式宣布创业间隔年
- **背景**: 退休后"很少再研究古典互联网"，对区块链和去中心化技术有长期兴趣（始于2013年比特币）。
- **动机**: "退休"后的空虚感+技术激情未灭。Web3的兴奋感"使得他即使在已经退休之后又重回第一线，再次点燃了作为开发者的激情"。他认为"Web3是下一代互联网，涵盖着金融和文化领域的价值互联网"。
- **过程**: 2022年1月在Mirror.xyz宣布创业间隔年。认识约30家VC的投资经理，最终搁置天使轮融资计划。创立Checks Finance和CodeforDAO两个项目，进入密集迭代周期。在Mirror上发表科幻小说《永不消逝的哈希》作为Web3写作实验。与SECBIT（安比实验室）合作从事零知识证明、形式化验证研究。
- **结果**: 2022年加密市场进入熊市，创业项目面临周期考验。他表示准备将大部分个人资产转换为以太坊，"All in Web3"。
- **本人反思**: 写作记录是他一以贯之的习惯："童年的日记、高中杂文、现在的推特记录。形式虽变，但记录与反思的本质未变。"
- **来源**: Mirror.xyz个人主页；PANews专访；CoinYuppie对话
- **可信度**: 一手（本人亲述 + 链上记录）

### 决策 9: 从东京搬至硅谷，转型风险投资人

- **时间**: 2024年8月
- **背景**: AI技术爆发，世界发生"根本性的变化"。退休5年后，34岁的他感到需要重新参与世界变革。
- **动机**: "世界在发生一些根本性的变化，他出于本能地想要参与进去。而这一次是以投资人的身份。"寻找"下一个浪头"。
- **过程**: 从东京搬到硅谷，以风险投资人身份参与AI领域创业和投资。热衷于利用AI Agent重构个人工作流。
- **结果**: 目前身份为"程序员、独立开发者及风险投资人"。2025年5月在东京银座接受独立记者采访讨论"算力霸权与人类的未来"。
- **本人反思**: 从最初认为财务自由等于绝对自由，到认识到这是一个"错误判断"。"最重要的是，我们需要在人生很早期的阶段意识到这个问题以及它的重要性。"
- **来源**: PANews专访（2025年10月发布）；学姿势网站访谈（2026年4月）
- **可信度**: 一手（本人亲述）

## 决策链内在逻辑

郭宇的决策链呈现出清晰的内在一致性：

1. **生存驱动** → 自学编程（被迫之选，但选择了自己的方式）
2. **能力兑现** → 支付宝/创业/字节跳动（把编程能力转化为职业回报）
3. **财务积累** → 期权+加密货币+股票+日本房产（多元被动收入体系）
4. **自由实现** → 退休/移居日本（被动收入>=工资时果断离场）
5. **意义追寻** → Web3创业/投资人转型（财务自由后寻找下一个"兴奋感"）

贯穿始终的信条："关注自己、理解自己到底想做什么。不管是学编程还是入职支付宝，在每一个时刻我做出决定的时候，都是做我想做的事情。"
